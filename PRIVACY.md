# Privacy Policy

**Phograph for Windows** is an open source visual dataflow programming environment for Windows.

## Data Collection

This app does **not** collect, store, or transmit any personal data.

## Network Usage

This app does not make any network connections. All project files are stored and processed locally.

## Local Storage

All data (project files, settings, preferences) is stored locally on your device. This data is not transmitted anywhere.

## Analytics

This app does not include any analytics, tracking, or advertising frameworks.

## Third-Party Services

This app does not use any third-party services that collect user data.

## Contact

If you have questions about this privacy policy, please open an issue at:
https://github.com/avwohl/phographw/issues

## Changes

This privacy policy may be updated occasionally. Changes will be posted to this repository.
